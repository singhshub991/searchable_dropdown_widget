# Searchable Dropdown Widget Example

This is an example Flutter app demonstrating how to use the `searchable_dropdown_widget` package.

## Running the Example

1. Make sure you have Flutter installed
2. Navigate to this directory:
   ```bash
   cd example
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## What's Demonstrated

- **SearchableDropdown**: Basic usage with search and filter functionality
- **SearchableTextFormField**: Form integration with validation
- Custom filtering and item display
- Form validation examples
- Real-time search and selection

## Features Shown

- Search countries by name
- Filter results in real-time
- Display selected item details
- Form validation
- Custom item builders
- Material Design 3 styling
